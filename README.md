# Superstore-Retail-Analysis

🛒 Superstore Sales Performance Dashboard

📊 Project Overview

This project transforms raw retail data into an interactive executive summary, enabling stakeholders to monitor KPIs across products, regions, and customer categories.

🚀 Key Features

End-to-End Retail KPI Tracking: Engineered a robust reporting tool for $2.26M in total sales across 4,922 orders, covering 49 U.S. states and 529 cities. 

Advanced Customer Segmentation: Leveraged data modeling to demonstrate that the Consumer segment drives 50.76% ($1.15M) of total revenue, followed by Corporate and Home Office segments.

Product Performance Optimization: Isolated top-performing assets, identifying the Technology category and the Canon imageCLASS 2200 ($61,599) as the leading revenue drivers. 

🛠️ Tech Stack

Tool:Power BI Desktop 

Data Source: Superstore Sales Dataset 

Key Skills: ETL (Power Query), Geographic Mapping, Trend Analysis
